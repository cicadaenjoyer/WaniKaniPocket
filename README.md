## Project Overview

Tabi is a dedicated mobile application designed to extend the comprehensive Japanese language learning experience of WaniKani to iOS and Android platforms.

This application aims to provide mobile users with the same user experience as the web version of the app.

-   Current Level and Progress Overview
-   Perform Lessons
-   Perform Review Quizzes
-   Review Radical, Kanji, and Vocabulary Subjects
